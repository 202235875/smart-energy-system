# Smart Energy Management System

## 프로젝트 소개
이 프로젝트는 특정 도시의 실제 날씨 데이터를 바탕으로 태양광 발전량과 건물의 전력 소비를 비교해보고, 에너지 자립이 어느 정도 가능한지 확인해보기 위해 만든 프로그램입니다.

도시를 입력하고 태양광 설비나 배터리 용량 같은 조건을 설정하면, 시간대별로 전기가 어떻게 생산되고 소비되는지를 계산해서 결과로 보여줍니다.

---

## 주요 기능
- 도시별 기상 데이터를 활용한 태양광 발전량 계산  
- 건물 유형과 시간대에 따른 전력 소비량 시뮬레이션  
- 배터리 충전 및 방전 흐름 반영  
- 자급률과 외부 전력 사용 비율 확인  
- 그래프를 통한 결과 시각화  

---

## 사용 방법
1. 도시 이름 입력 (예: Seoul, Tokyo, Dubai)  
2. 태양광 용량, 배터리 용량 등 원하는 값으로 설정  
3. ‘분석 시작’ 버튼 클릭  
4. 결과 그래프와 지표 확인
